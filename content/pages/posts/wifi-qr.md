## Wifi QR Code

- Code Format
  ```
  WIFI:S:<SSID>;T:<WEP|WPA|blank>;P:<PASSWORD>;H:<true|false|blank>;;
  ```
- example
  ```
  WIFI:S:DGKIM5G;T:WPA2;P:secretWiFi@@;;
  ```
